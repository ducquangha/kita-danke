# Kita-Danke-Seite auf GitHub Pages veröffentlichen (5 Minuten)

1. **GitHub-Konto**: Auf https://github.com einloggen (oder kostenlos registrieren).
2. **Neues Repository**: Oben rechts auf **+** → **New repository**.
   - Name: `kita-danke`
   - **Public** auswählen → **Create repository**
3. **Dateien hochladen**: Im neuen Repo auf **uploading an existing file** klicken.
   - Die ZIP-Datei vorher entpacken und den **Inhalt** des Ordners `kita-danke` hineinziehen
     (also `index.html`, den Ordner `amelie` und den Ordner `audio`).
   - Unten auf **Commit changes** klicken.
4. **Pages aktivieren**: Im Repo auf **Settings** → links **Pages**.
   - Unter "Branch": `main` auswählen, Ordner `/ (root)` → **Save**.
5. **Fertig**: Nach 1–2 Minuten ist die Seite live unter:
   `https://DEIN-BENUTZERNAME.github.io/kita-danke/amelie/`

## Weitere Erzieherinnen hinzufügen

Für jede weitere Erzieherin einfach einen neuen Ordner (z. B. `maria/`) mit einer
`index.html` und die passende MP3 in `audio/` hochladen — sag mir einfach Name +
Sprachnachrichten, ich baue die Seiten und QR-Codes fertig.

## Wichtig

- Der QR-Code kodiert die URL fest (statisch) — er läuft **nie ab**, solange das
  Repo existiert. Also: Repo nicht umbenennen oder löschen.
- GitHub Pages ist dauerhaft kostenlos für öffentliche Repos.
